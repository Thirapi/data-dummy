API Documentation

Resources

The API provides the following resources:

  • /users - List of all users
  • /posts - List of all posts

Routes

Use the following routes to interact with the API:

  • /users - Retrieve all users
  • /users/:id - Retrieve a user by ID (e.g., /users/1)
  • /posts - Retrieve all posts
  • /posts/:id - Retrieve a post by ID (e.g., /posts/1)

How to Use the API

To use the API, simply send a GET request to the desired endpoint. Here are some examples:

Example 1: Get all users

GET /users

URL: http://localhost:3000/users

Example 2: Get a user by ID

GET /users/1

URL: http://localhost:3000/users/1

Example 3: Get all posts

GET /posts

URL: http://localhost:3000/posts

Example 4: Get a post by ID

GET /posts/1

URL: http://localhost:3000/posts/1
