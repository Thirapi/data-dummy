api-dummy

Ini adalah dokumentasi untuk API Dummy yang menyediakan data user dan post. Untuk sementara hanya mendukung metode GET dan belum mendukung metode lain seperti POST, PUT, atau DELETE

Berikut adalah informasi tentang resource dan rute yang tersedia.

Resource

API ini menyediakan resource berikut.

URL base untuk API ini adalah https://data-dummy.vercel.app/:

  • /users - memiliki 30 data users
  • /posts - memiliki 30 data posts

Rute

Gunakan rute berikut untuk berinteraksi dengan API:

  • /users - Mengambil semua pengguna
  • /users/:id - Mengambil pengguna berdasarkan ID (misalnya, /users/1)
  • /posts - Mengambil semua postingan
  • /posts/:id - Mengambil postingan berdasarkan ID (misalnya, /posts/1)

Cara Menggunakan API

Untuk menggunakan API, kirimkan permintaan GET ke endpoint yang diinginkan. Berikut adalah beberapa contoh:

Contoh 1: Mengambil semua pengguna

GET /users

URL: https://data-dummy.vercel.app/users

Contoh 2: Mengambil pengguna berdasarkan ID

GET /users/1

URL: https://data-dummy.vercel.app/users/1

Contoh 3: Mengambil semua postingan

GET /posts

URL: https://data-dummy.vercel.app/posts

Contoh 4: Mengambil postingan berdasarkan ID

GET /posts/1

URL: https://data-dummy.vercel.app/posts/1
