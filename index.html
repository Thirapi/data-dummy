api-dummy

Ini adalah dokumentasi untuk API Dummy yang menyediakan data user dan post. Sekarang telah mendukung metode GET, POST, PUT, PATCHdan DELETE

Berikut adalah informasi tentang resource dan rute yang tersedia.

Resource

API ini menyediakan resource berikut.

URL base untuk API ini adalah https://data-dummy.vercel.app/:

  • /users - memiliki 30 data users
  • /posts - memiliki 30 data posts

Rute

Gunakan rute berikut untuk berinteraksi dengan API:

  • GET - /users - Mengambil semua pengguna
  • GET - /posts - Mengambil semua postingan
  • GET - /users/:id - Mengambil pengguna berdasarkan ID (misalnya, /users/1)
  • GET - /posts/:id - Mengambil postingan berdasarkan ID (misalnya, /posts/1)
  • POST - /users - Menambah user baru
  • POST - /posts - Menambah postingan baru
  • PUT - /users/:id - Mengganti seluruh data postingan berdasarkan ID (misalnya, /users/1)
  • PUT - /posts/:id - Mengganti seluruh data postingan berdasarkan ID (misalnya, /posts/1)
  • PATCH - /users/:id - Memperbarui sebagian data postingan berdasarkan ID (misalnya, /users/1)
  • PATCH - /posts/:id - Memperbarui sebagian data postingan berdasarkan ID (misalnya, /posts/1)
  • DELETE - /users/:id - Menghapus postingan berdasarkan ID (misalnya, /users/1)
  • DELETE - /posts/:id - Menghapus postingan berdasarkan ID (misalnya, /posts/1)
